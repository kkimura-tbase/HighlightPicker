# LexiLight OCR

黄色でハイライトした英語教材やPDFのスクリーンショットから、ハイライト部分の英単語・熟語をOCRで抽出し、文脈や日本語訳と一緒に保存して復習できるWebアプリです。

## できること

- 画像の選択、ドラッグ&ドロップ、貼り付けで読み込み
- Canvas APIによる黄色ハイライト領域の検出
- Tesseract.jsによるブラウザ内OCR
- 抽出結果の単語、品詞、意味、例文、日本語訳の確認と修正
- Google Apps Script Webアプリ経由でGoogleスプレッドシートに保存
- ブラウザのローカル保存による簡易利用
- 単語一覧の検索、並び替え、削除、CSV出力
- 保存済み単語からのランダム復習
- 利用者ごとの履歴管理（生徒1、生徒2、教師1、教師2）

## 使い方

1. `index.html` をブラウザで開きます。GitHub Pagesなどの静的ホスティングにも配置できます。
2. 初回表示時に、氏名または任意のIDを登録します。
3. GAS連携は `app.js` の `GAS_ENDPOINT` に設定された固定URLを使います。
4. 黄色でハイライトした英文画像を読み込みます。
5. 必要に応じて読み取りエリアやハイライト検出感度を調整します。
6. 「抽出する」を押し、結果を確認・修正してから「単語帳に追加」で保存します。

GAS URLを入力しない場合でも、ブラウザのローカル保存だけで利用できます。ただし、別端末との同期や自動翻訳は利用できません。

## Google Apps Script の設定

1. Googleスプレッドシートを新規作成します。
2. 「拡張機能」から「Apps Script」を開きます。
3. このリポジトリの `Code.gs` の内容をApps Scriptの `Code.gs` に貼り付けます。
4. 「デプロイ」から「新しいデプロイ」を選びます。
5. 種類を「ウェブアプリ」にします。
6. 実行ユーザーとアクセス権は利用環境に合わせて設定します。
7. 発行されたWebアプリURLを、このアプリの「GAS WebアプリURL」に入力します。

## OCR精度を上げるコツ

- 文字が小さい場合は、ブラウザやPDFビューアで拡大してからスクリーンショットを撮ります。
- ハイライト色は薄すぎない黄色にします。
- 画像の傾き、影、圧縮ノイズを減らすと抽出が安定します。
- 余計な黄色が多い画像では、読み取りエリアを指定します。

## ファイル構成

- `index.html`: アプリ画面
- `styles.css`: 画面デザイン
- `app.js`: OCR、ハイライト検出、保存、一覧、復習、CSV出力
- `Code.gs`: Google Apps Script用Webアプリ。保存、自動翻訳、スプレッドシート操作


## 利用者の登録方法

初回起動時に、氏名または任意のIDを入力します。入力した値はそのブラウザのローカル保存に残り、以後は同じ利用者として履歴を保存・表示します。

共有端末で利用者を変更したい場合は、ブラウザのサイトデータまたは localStorage の `context-vocab-owner` を削除してから開き直します。教師側で端末ごとに強く固定したい場合は、`app.js` の `FIXED_OWNER` に任意の氏名またはIDを入れます。
