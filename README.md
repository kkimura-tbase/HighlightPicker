# Context Vocab OCR

黄色でハイライトした英語長文の画像から、ハイライト部分の単語をOCRで抽出し、文脈・推測メモと一緒に保存して復習する静的Webアプリです。

## できること

- 画像アップロード、ドラッグ＆ドロップ、スクリーンショット貼り付け
- Canvas APIによる黄色ハイライト領域の検出
- Tesseract.jsによるOCR
- 抽出された単語・英文・推測メモの確認と修正
- ハイライト単語の日本語訳と、英文全体の日本語訳の自動表示
- GAS Webアプリ経由でGoogleスプレッドシートに保存
- 単語一覧表示、検索、ランダム復習クイズ
- CSVエクスポート

## フロントエンドの使い方

`index.html` をブラウザで開くか、GitHub Pagesなどの静的ホスティングに配置します。

GAS連携を使う場合は、アプリ画面の「GAS WebアプリURL」に公開済みWebアプリの `/exec` URLを入力してください。未入力でも端末内のブラウザ保存領域に一時保存できます。

日本語訳の自動表示にはGAS側の `LanguageApp.translate` を使います。GAS WebアプリURLを設定してからOCR抽出すると、単語の日本語訳と英文の日本語訳が自動で入力されます。

## GASの設定

1. Googleスプレッドシートを新規作成します。
2. 「拡張機能」→「Apps Script」を開きます。
3. `gas/Code.gs` の中身をApps Scriptの `Code.gs` に貼り付けます。
4. Apps Scriptの「デプロイ」→「新しいデプロイ」を選びます。
5. 種類を「ウェブアプリ」にします。
6. 実行ユーザーを「自分」、アクセスできるユーザーを利用環境に合わせて設定します。
7. 発行されたWebアプリURLを、このアプリの入力欄に入れます。

## OCRの精度を上げるコツ

- 文字が小さい場合は、ブラウザやPDFビューアで拡大してからスクリーンショットを撮ります。
- ハイライト色は薄すぎない黄色にします。
- 画像内の傾きや影が少ないほど、単語と英文の抽出が安定します。

## ファイル構成

- `index.html`: アプリ画面
- `styles.css`: 画面デザイン
- `app.js`: OCR、黄色検出、保存、一覧、復習、CSV出力
- `gas/Code.gs`: Google Apps Script用Webアプリ、スプレッドシート保存、自動翻訳
